# image2ascii  
## C
only accept bmp file  
##### usage:  
```
make
./image2ascii -p <fileName> -w <width> [-i] [-h]
```
ex:
```
./image2ascii -p shiki.png -w 40 -i
```
___
## C++
accept all format support by opencv  
##### usage:
```
make
./im2ascii_cpp <fileName> <width> <inverse>
```
